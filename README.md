# Henry Lo — Portfolio

Personal portfolio website showcasing my work as a Full-Stack Software Engineer specializing in Next.js, NestJS, and Django.

## Tech Stack

- **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript
- **Icons**: Font Awesome 6
- **Hosting**: GitHub Pages

## Features

- Responsive design with mobile-first approach
- Three theme modes (dark, light, oled) with persistence
- Respects OS-level `prefers-color-scheme` preference
- Accessible navigation with skip-to-content link
- Animated mind-map style interests section

## Local Development

Simply open `index.html` in a browser, or use a local dev server:

```bash
npx serve .
```
